# Backend 2.2 — Node.js Course

Welcome to the **Backend 2.2** course repository. This repo contains lesson materials, code examples, and exercises organized by topic.

## Repository Structure

```
[2.2] nodejs-backend-course/
├── 01-built-in-modules/     # path, fs, os, events, http, etc.
├── 02-coming-soon/           # (future lessons will be added here)
└── README.md
```

Each lesson folder contains:
- `README.md` — Topic overview, key concepts, and references
- `examples/` — Code examples demonstrated during the lesson
- `exercises/` — Practice exercises for students

## Lessons

| #  | Topic             | Description                                      |
|----|-------------------|--------------------------------------------------|
| 01 | Built-in Modules  | `path`, `fs`, `os`, `events`, `http`, and more   |

> More lessons will be added as the course progresses.

## Getting Started

1. Clone this repository:
   ```bash
   git clone <repo-url>
   ```
2. Navigate to a lesson folder
3. Run `npm install` if the lesson has a `package.json`
4. Follow the lesson's `README.md`

## Requirements

- Node.js (v18 or later recommended)
- A code editor (VS Code recommended)
